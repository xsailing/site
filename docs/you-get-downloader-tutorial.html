        # You-get Downloader Tutorial

        # Tutorial

        1. Launch “You-get Downloader” App.
        2. Go to some website such as You-tube, view some video you want, and copy the URL like “https://www.youtube.com/watch?v=R5NC9yyTeZ8”.
        3. Paste URL above to “Video URL” input box.
        4. Choose/change storage location you want to store the video.
        5. Click “Start” button.
        6. Wait a moment, choose definition you want to download.
        7. Wait for download to complete, then go to storage location to check your video.

        You may encounter network problems. Check your proxy config and try again may help you.
